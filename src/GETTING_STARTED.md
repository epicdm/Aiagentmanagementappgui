## ğŸ†˜ Troubleshooting

### Can't See the Dashboard?
**Problem:** Stuck on landing or auth page

**Solution:**
1. Make sure you clicked "Get Started"
2. Complete sign in/sign up
3. Wait for authentication to complete
4. Dashboard should load automatically

### No Data on Dashboard?
**Problem:** Everything shows "0"

**Solution:**
1. Create your first agent (click the button!)
2. Make some test calls
3. Wait a few seconds for data to refresh
4. Refresh the page if needed

### Can't Create Agent?
**Problem:** Create button doesn't work

**Solution:**
1. Fill in all required fields (marked with *)
2. Check internet connection
3. Look for error messages
4. Try refreshing and retry

### Live Calls Not Showing?
**Problem:** No calls appear on Live Calls page

**Solution:**
1. Make sure agents are active
2. Verify phone numbers are assigned
3. Check if any calls are actually in progress
4. Page auto-refreshes every 2 seconds
